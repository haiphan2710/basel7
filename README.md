## About Base Laravel 7
Base on Laravel 7

- We have already features:
  + Authentication

## Install BaseL7

- For installing BaseL7, run command: php artisan basel7:install

## Create a model

- Commands:
  + Model: php artisan basel7:model ModelName
  + Pivot: php artisan basel7:model Models/{ModelName} --pivot
